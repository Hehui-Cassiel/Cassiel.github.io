搭建hexo博客

申请github账号

Github注册页面

输入用户名，Email，密码，注册账号

创建博客仓库

注意，仓库名应该为：用户名.github.io。比如，我的仓库名是：hehui-cassiel.github.io。

搭建环境

1.安装下载node.js

2.安装npm

3.检验node和npm是否安装成功，在cmd中输指令：node-v 和 npm -v

搭建hexo和git

安装HEXO

使用安装语句，并检验是否成功

1.npm install -g hexo-cli

2.hexo init blog

3.hexo -v

建站

1.在终端切换（cd）一个你想指定的目录(blog)，执行下面这个命令
    hexo init blog

2.在这个目录下会出现你刚刚创建的blog文件夹，切换到（cd）blog文件夹执行以下命令来安装npm
    npm install

③执行以下命令来开启服务器
   hexo server

④开启后访问[http://localhost:4000/

](http://localhost:4000/)

与github联系起来

1.首先创造一个新的仓库，给仓库命名（示例：everything**.github,io**，其中everything可改变，加粗地方不可变）

2.将blog名下的_config.yml内容那个最后更改为：

deploy:

    type:git

    repository:https://github.com/username/username.github.io.git

3.在blog目录下执行命令
//生成静文件 hexo generate
//部署网站 hexo deploy

此时可以访问属于自己的网站了https://username.github.io/

建立博客后怎样发布

在本地文件中，用Git Bash执行
hexo new "filename"
hexo g
hexo s
hexo d
1.hexo g #完整命令为hexo generate,用于生成静态文件
2.hexo s #完整命令为hexo server,用于启动服务器，主要用来本地预览
3.hexo d #完整命令为hexo deploy,用于将本地文件发布到github等git仓库上
4.hexo n “my article” #完整命令为hexo new,用于新建一篇名为“filename”的文章
